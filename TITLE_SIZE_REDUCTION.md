# Title Header Size Reduction - Site-Wide

## Overview
All title headers across the website have been reduced by approximately 15-21% to create a more refined, modern, and balanced appearance.

## Changes Made

### Desktop Title Sizes

#### **Main Section Titles** (`.section-title`)
- **Before**: 2.8rem
- **After**: **2.2rem**
- **Reduction**: 21%
- Used on: Focus Areas, Programs, News, etc.

#### **Hero Titles** (`.hero-title`)
- **Before**: 2.4rem
- **After**: **2rem**
- **Reduction**: 17%
- Used on: Homepage hero, page headers

#### **Large Page Titles** (3rem titles)
Reduced from 3rem to 2.4rem (20% reduction):
- **Register Title** (`.register-title`)
- **FAQ Hero Title** (`.faq-hero-title`)
- **Blog Hero Title** (`.blog-hero-title`)

### Mobile Title Sizes

Mobile breakpoints already have optimized smaller sizes:

#### **768px Breakpoint**
- Hero title: 1.6rem
- Section title: Proportionally reduced

#### **480px Breakpoint**
- Hero title: 1.35rem
- Section title: 1.5rem
